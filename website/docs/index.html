ANZX AI Platform API Documentation

Welcome to the ANZX AI Platform API documentation. Our REST API allows you to create, manage, and interact with AI assistants programmatically.

🚀 Base URL

https://api.anzx-ai.com/v1

Features

  • RESTful API design with predictable URLs
  • JSON request and response bodies
  • Standard HTTP response codes
  • Comprehensive error messages
  • Rate limiting and authentication
  • Webhook support for real-time events

Quick Start

Get up and running with the ANZX AI Platform API in minutes.

1. Get Your API Key

First, sign up for an account and get your API key from the dashboard:

Dashboard → Settings → API Keys 
# Your API key will look like this:
anzx_sk_1234567890abcdef...

2. Make Your First Request

Test your API key by fetching your account information:

curl -X GET "https://api.anzx-ai.com/v1/auth/me" \
  -H "Authorization: Bearer anzx_sk_1234567890abcdef..." \
  -H "Content-Type: application/json"
const response = await fetch('https://api.anzx-ai.com/v1/auth/me', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer anzx_sk_1234567890abcdef...',
    'Content-Type': 'application/json'
  }
});

const user = await response.json();
console.log(user);
import requests

headers = {
    'Authorization': 'Bearer anzx_sk_1234567890abcdef...',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.anzx-ai.com/v1/auth/me', headers=headers)
user = response.json()
print(user)

3. Create Your First Assistant

Create an AI assistant to handle customer support:

curl -X POST "https://api.anzx-ai.com/v1/assistants" \
  -H "Authorization: Bearer anzx_sk_1234567890abcdef..." \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Support Assistant",
    "type": "support",
    "description": "Handles customer support inquiries",
    "capabilities": ["communication", "technical", "billing"]
  }'
const assistant = await fetch('https://api.anzx-ai.com/v1/assistants', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer anzx_sk_1234567890abcdef...',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Support Assistant',
    type: 'support',
    description: 'Handles customer support inquiries',
    capabilities: ['communication', 'technical', 'billing']
  })
});

const result = await assistant.json();
console.log(result);
import requests

data = {
    'name': 'Support Assistant',
    'type': 'support',
    'description': 'Handles customer support inquiries',
    'capabilities': ['communication', 'technical', 'billing']
}

response = requests.post(
    'https://api.anzx-ai.com/v1/assistants',
    headers=headers,
    json=data
)

assistant = response.json()
print(assistant)

Authentication

The ANZX AI Platform API uses API keys for authentication. Include your API key in the Authorization header:

Authorization: Bearer anzx_sk_1234567890abcdef...

⚠️ Keep Your API Key Secure

Never expose your API key in client-side code or public repositories. Store it securely as an environment variable.

API Key Types

Type Prefix Usage
Secret Key anzx_sk_ Server-side API calls
Publishable Key anzx_pk_ Client-side widget integration

Assistants

Create and manage AI assistants with specialized capabilities.

POST /assistants

Create a new AI assistant

Request Body

{
  "name": "Support Assistant",
  "type": "support",
  "description": "Handles customer support inquiries",
  "capabilities": ["communication", "technical", "billing"],
  "system_prompt": "You are a helpful customer support assistant...",
  "knowledge_sources": ["kb_123", "kb_456"]
}

Response

{
  "id": "asst_1234567890",
  "name": "Support Assistant",
  "type": "support",
  "description": "Handles customer support inquiries",
  "capabilities": ["communication", "technical", "billing"],
  "status": "active",
  "created_at": "2024-01-15T10:30:00Z",
  "updated_at": "2024-01-15T10:30:00Z"
}
GET /assistants

List all assistants

Query Parameters

Parameter Type Description
limit integer Number of results to return (default: 20, max: 100)
offset integer Number of results to skip (default: 0)
type string Filter by assistant type

Messages

Send and receive messages in conversations.

POST /conversations/{conversation_id}/messages

Send a message in a conversation

Request Body

{
  "content": "Hello, I need help with my account",
  "role": "user",
  "metadata": {
    "source": "chat_widget",
    "user_agent": "Mozilla/5.0..."
  }
}

Response

  "id": "msg_1234567890",

Chat Widget Integration

Embed the ANZX AI chat widget on your website for seamless customer interactions.

Basic Integration

Add the widget to your website with minimal configuration:

HTML 
<!-- Add this to your HTML -->
<div id="anzx-chat-widget"></div>

<script>
  window.anzxChatConfig = {
    organizationId: 'your-org-id',
    assistantId: 'your-assistant-id',
    theme: 'blue',
    position: 'bottom-right'
  };
</script>
<script src="https://cdn.anzx-ai.com/widget.js"></script>

Configuration Options

Option Type Description Default
organizationId string Your organization ID Required
assistantId string Assistant to use for conversations Required
theme string Color theme (blue, green, purple, dark) 'blue'
position string Widget position (bottom-right, bottom-left) 'bottom-right'
title string Widget title 'Chat with us'

Custom Styling

Customize the widget appearance with CSS:

CSS 
/* Custom widget styles */
.anzx-chat-widget {
  --primary-color: #your-brand-color;
  --border-radius: 12px;
  --font-family: 'Your Font', sans-serif;
}

.anzx-chat-widget .chat-header {
  background: linear-gradient(45deg, #your-color1, #your-color2);
}

Example: Support Bot

Complete example of creating a customer support bot with knowledge base integration.

// Complete support bot setup
const ANZX_API_KEY = 'anzx_sk_your_key_here';
const BASE_URL = 'https://api.anzx-ai.com/v1';

class SupportBot {
  constructor(apiKey) {
    this.apiKey = apiKey;
    this.headers = {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    };
  }

  async createAssistant() {
    const response = await fetch(`${BASE_URL}/assistants`, {
      method: 'POST',
      headers: this.headers,
      body: JSON.stringify({
        name: 'Customer Support Bot',
        type: 'support',
        description: 'Handles customer inquiries and support tickets',
        capabilities: ['communication', 'technical', 'billing'],
        system_prompt: `You are a helpful customer support assistant. 
                       Always be polite, professional, and try to resolve 
                       issues quickly. Escalate to human agents when needed.`
      })
    });
    
    return await response.json();
  }

  async startConversation(assistantId, userContext) {
    const response = await fetch(`${BASE_URL}/conversations`, {
      method: 'POST',
      headers: this.headers,
      body: JSON.stringify({
        assistant_id: assistantId,
        title: 'Support Inquiry',
        channel: 'chat',
        user_context: userContext
      })
    });
    
    return await response.json();
  }

  async sendMessage(conversationId, message) {
    const response = await fetch(`${BASE_URL}/conversations/${conversationId}/messages`, {
      method: 'POST',
      headers: this.headers,
      body: JSON.stringify({
        content: message,
        role: 'user'
      })
    });
    
    return await response.json();
  }
}

// Usage example
const bot = new SupportBot(ANZX_API_KEY);

// Create assistant
const assistant = await bot.createAssistant();
console.log('Assistant created:', assistant.id);

// Start conversation
const conversation = await bot.startConversation(assistant.id, {
  user_id: 'user_123',
  email: 'customer@example.com',
  name: 'John Doe'
});

// Send message
const response = await bot.sendMessage(conversation.id, 'I need help with my billing');
console.log('AI Response:', response.ai_response.content);
import requests
import json

class SupportBot:
    def __init__(self, api_key):
        self.api_key = api_key
        self.base_url = 'https://api.anzx-ai.com/v1'
        self.headers = {
            'Authorization': f'Bearer {api_key}',
            'Content-Type': 'application/json'
        }
    
    def create_assistant(self):
        data = {
            'name': 'Customer Support Bot',
            'type': 'support',
            'description': 'Handles customer inquiries and support tickets',
            'capabilities': ['communication', 'technical', 'billing'],
            'system_prompt': '''You are a helpful customer support assistant. 
                               Always be polite, professional, and try to resolve 
                               issues quickly. Escalate to human agents when needed.'''
        }
        
        response = requests.post(
            f'{self.base_url}/assistants',
            headers=self.headers,
            json=data
        )
        
        return response.json()
    
    def start_conversation(self, assistant_id, user_context):
        data = {
            'assistant_id': assistant_id,
            'title': 'Support Inquiry',
            'channel': 'chat',
            'user_context': user_context
        }
        
        response = requests.post(
            f'{self.base_url}/conversations',
            headers=self.headers,
            json=data
        )
        
        return response.json()
    
    def send_message(self, conversation_id, message):
        data = {
            'content': message,
            'role': 'user'
        }
        
        response = requests.post(
            f'{self.base_url}/conversations/{conversation_id}/messages',
            headers=self.headers,
            json=data
        )
        
        return response.json()

# Usage example
bot = SupportBot('anzx_sk_your_key_here')

# Create assistant
assistant = bot.create_assistant()
print(f"Assistant created: {assistant['id']}")

# Start conversation
conversation = bot.start_conversation(assistant['id'], {
    'user_id': 'user_123',
    'email': 'customer@example.com',
    'name': 'John Doe'
})

# Send message
response = bot.send_message(conversation['id'], 'I need help with my billing')
print(f"AI Response: {response['ai_response']['content']}")
